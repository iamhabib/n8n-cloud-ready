# n8n

n8n is an extendable workflow automation tool that enables you to connect various services and automate tasks with ease. It provides a visual interface for designing workflows, integrating APIs, and orchestrating complex business logic without writing extensive code.

### Docker

You can also run n8n using Docker:

```bash
docker-compose up -d
```

## Usage

- Access the web UI at [http://localhost:5678](http://localhost:5678)
